Question 5: Difference between block and inline

Block Elements Inline Elements
Block elements are those elements that always starts with a new line. Inline elements are those that doesnt start with a new line
Block Elements occupy the full width Inline elements occupy only sufficient width required.
Block elements doesnt allow other elements to sit behind Inline elements allow other inline elements to sit behind.
